CLI not installed

Hot Sheet needs to be launched from your project directory so it knows where to store data.

Click below to install the hotsheet command, or copy the command manually:

CLI is installed

Open a terminal, navigate to your project, and run:

cd /path/to/your/project
hotsheet

This opens Hot Sheet with your project's data directory. Close this window — the app will reopen from the terminal.

CLI installed successfully

Now open a terminal, navigate to your project, and run:

cd /path/to/your/project
hotsheet
